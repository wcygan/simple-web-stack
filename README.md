# Simple Web Stack

|---|---|

| Backend | [Axum](https://docs.rs/axum/latest/axum/) |

| Cache | [DragonflyDB](https://www.dragonflydb.io/) + [redis-rs](https://github.com/redis-rs/redis-rs) |
